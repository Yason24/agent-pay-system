# ✅ Шаг 1 — Создай Password Hash Service

Создай файл:

```
app/Services/HashService.php
```

```php
<?php

namespace App\Services;

class HashService
{
    public function make(string $password): string
    {
        return password_hash($password, PASSWORD_BCRYPT);
    }

    public function verify(string $password, string $hash): bool
    {
        return password_verify($password, $hash);
    }
}
```

# ✅ Шаг 2 — Зарегистрируй сервис в Container

У тебя framework работает через Service Providers.

Создай (если ещё нет):

```
app/Providers/AppServiceProvider.php
```

```php
use App\Services\HashService;

$this->app->singleton(HashService::class, function () {
    return new HashService();
});
```

(или добавь в существующий provider)

# ✅ Шаг 3 — Генерация пароля при регистрации

В твоём AuthController:

```php
use App\Services\HashService;
use App\Models\User;

class AuthController
{
    public function register(HashService $hash)
    {
        $password = $hash->make($_POST['password']);

        User::create([
            'email' => $_POST['email'],
            'password' => $password,
        ]);
    }
}
```

💥 Всё.
DI сам подставит сервис.

# ✅ Шаг 4 — Проверка при логине

```php
public function login(HashService $hash)
{
    $user = User::where('email', $_POST['email'])->first();

    if (!$user) {
        die('User not found');
    }

    if (!$hash->verify($_POST['password'], $user->password)) {
        die('Wrong password');
    }

    echo "LOGIN OK";
}
```

## Сделай Seeder (идеально для dev)

```
seeders/UserSeeder.php
```

```php
use App\Services\HashService;
use App\Models\User;

class UserSeeder
{
    public function run(PDO $db)
    {
        $hash = new HashService();

        User::create([
            'email' => 'admin@test.com',
            'password' => $hash->make('12345'),
        ]);
    }
}
```

Запуск:

```bash
php console db:seed UserSeeder
```

✅ пароль = `12345`
✅ в базе хранится hash
✅ архитектура правильная
